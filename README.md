# pet-shelter-api

Pet shelter api exposes three end points:
 1. Index of pets
 2. Pet lookup by id
 3. Pet creation
 
 
 Setup on local machine:
  Prerequisites: 
    Node js installed
  1. Clone or download the repository
  2. Commands to run when you are in cloned directory
  
     2.1 pet-shelter-api> npm install
     2.2 pet-shelter-api> node bin/www.js
     This will start the server at localhost:8000
